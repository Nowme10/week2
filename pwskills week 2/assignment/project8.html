HTML Text Formatting

    1. Strong

    • The strong tag is used to define text with strong importance
    • The content inside is typically displayed in bold.
    • Another accepted use for strong is to denote the labels of paragraphs which represent notes or warnings within the text of a page.

    2. Mark

    • When used in a quotation or block quote.
    • it generally indicates text which is of special interest but is not marked in the original source material,
    • Think of this like using a highlighter pen in a book to mark passages that you find of interest.

    3. Del

    • The del HTML element represents a range of text that has been deleted from a document
    • This can be used when rendering "track changes" or source code diff information, for example
    • The ins element can be used for the opposite purpose: to indicate text that has been added to the document.
